**Projeto Realizado no terceiro dia do Bootcamp**

Projeto realizado no quarto dia de bootcamp, realizamos um case para a criação de um programa de gerenciamento de chamados.

**Case3.1**
Criação de um mini Service Desk de TI, permitindo criar, acompanhar e gerenciar chamados. A estrututa do projeto deixei de forma detalhada abaixo:

**Estrutura da solução**
- O sistema foi dividido em camadas lógicas, cada uma com uma responsabilidade bem definida:

- Entidades: representam o domínio do problema (Chamado).

- Enums: padronizam valores como Status e Prioridade.

- Repositório: abstrai o armazenamento dos chamados.

- Serviços/Regras: controlam validações e transições de status.

- Console (UI): interação com o usuário final.
