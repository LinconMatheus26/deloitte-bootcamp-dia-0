**Repositório de Projetos – Organização por Branches**

Este repositório contém os projetos desenvolvidos durante o Bootcamp .Net, organizados de forma para separar as entregas de cada dia.

**Organização das branchs**

- Cada branch representa um dia do bootcamp.

*Estrutura foi adotada:*

- main(branch principal)

- dtt/dia1/feature/bootcamp

Projetos e atividades desenvolvidas no Dia 1.

- dtt/dia2/feature/bootcamp

Projetos e atividades desenvolvidas no Dia 2.

- dtt/dia3/feature/bootcamp

Projetos e atividades desenvolvidas no Dia 3.


**Para acessar os projetos, siga o passo a passo abaixo:**

- Clique em Branches

- Selecione o dia desejado (dtt/diaX/feature/bootcamp)

- Explore os projetos desenvolvidos naquele dia
